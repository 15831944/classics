# beginning with a hash '#' character and blank lines are ignored.

# Example entry:
#     is|a,us,um

# Explanation:

# is successful then all comma-separated entries are tried in turn.  E.g., for
# the example given above, if the word supplied is 'portis', then the possible
# alternates given are 'porta', 'portus', 'portum'.
